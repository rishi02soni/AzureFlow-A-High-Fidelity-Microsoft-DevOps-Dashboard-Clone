## 🧠 About The Project

**AzureFlow** is a high-fidelity clone inspired by Microsoft’s Azure DevOps dashboard.  
It recreates the complete sprint lifecycle experience — from authentication to task analytics — with a focus on clean UI, performance, and real-time insights.

This project dives deep into:

- State management  
- Interactive UI design  
- Real-time data visualization  
- Enterprise-style dashboard architecture
